# Интерпретация графиков по методологии ЯМР (Плетень О.И.)

Документ составлен по тексту **methodology.pdf** («Определение давности создания реквизитов документа, содержащего целлюлозу, методом импульсной ЯМР спектроскопии»).

---

## 1. Что за сигнал на графике

- **Ось X** — время, **микросекунды** (мкс). Это время после возбуждающего импульса.
- **Ось Y** — интенсивность сигнала (в интерфейсе прибора может быть в %, мВ или отн. ед.).

График отображает **спад свободной индукции (ССИ)** — затухание ЯМР-сигнала во времени.

---

## 2. Три линии: синяя, зелёная, красная

В методике (раздел 6.4, после формул для K и Pr) явно указано:

> **«Зеленая и синяя линии — это реальная и мнимая составляющие сигнала, красная (результирующая) — корень квадратный из суммы квадратов этих сигналов, голубым цветом на ней выделены те участки, которые используются в расчетах».**

Итого:

| Линия   | Смысл |
|--------|--------|
| **Синяя**  | Одна из квадратурных составляющих (реальная или мнимая часть сигнала). |
| **Зелёная**| Вторая квадратурная составляющая (мнимая или реальная). |
| **Красная**| **Результирующая** амплитуда: √(реальная² + мнимая²). Это «основной» наблюдаемый сигнал. |
| **Голубые участки на красной** | Фрагменты кривой, которые используются в расчётах (разложение на короткую и длинную компоненты). |

То есть **три линии — это не три разных образца и не три прокола**. Это один и тот же ССИ, показанный в виде двух квадратурных каналов и их модуля. Квадратурное детектирование используется для повышения чувствительности прибора.

---

## 3. Как из сигнала получают K и Pr

Методика опирается на **двухкомпонентную** модель релаксации ССИ:

- **Короткая (быстрорелаксирующая) компонента** — время релаксации T2М, макс. амплитуда **Ак** (мВ).
- **Длинная (медленнорелаксирующая) компонента** — время релаксации T2, макс. амплитуда **АД** (мВ).

Расчёт по методике:

- **Индекс кристалличности K** считается по формуле из алгоритма «RELAX 8SB45» по величинам T2М, T2, Ак, АД (разложение ССИ на короткую и длинную компоненты).
- **Плотность протонов Pr** считается как **Pr = Ак / m**, где m — масса навески (г). То есть Pr определяется **максимальной амплитудой короткой компоненты** и массой образца.

Эти расчёты выполняет ЯМР-анализатор (ПО «RELAX 8SB45»). В экспорте/интерфейсе они уже даны как **индекс кристалличности** и **протонная плотность** — те самые числа, которые мы видим в заголовке графика (crystallinity_index, proton_density). **По трём линиям вручную K и Pr не пересчитывают** — они уже результат обработки той же кривой (по сути, красной результирующей и выделенным участкам).

---

## 4. Участки «без покрытия» и «с покрытием»

По методике для каждого исследуемого реквизита берут два типа участков:

- **Участок без покрытия** (открытый) → даёт **K1**, **Pr1**.
- **Участок с покрытием** (под красящим веществом штриха) → даёт **K2**, **Pr2**.

Относительное изменение параметров целлюлозы характеризуют отношения K2/K1 и Pr2/Pr1.  
Файл **graphics_without_coverage.pdf** по названию соответствует графикам с **участков без покрытия**. Аналогичный файл «с покрытием» тогда даёт участки с покрытием. Для полного расчёта по методике нужны пары (K1, Pr1) и (K2, Pr2), т.е. данные и без покрытия, и с покрытием.

---

## 5. Формула G и давность D (по методике)

**Формула (1)** методики:

**G = K2/K1 + Pr2/Pr1**

Используются только **K** и **Pr** (индекс кристалличности и плотность протонов) для пары «участок с покрытием / участок без покрытия». Отдельных величин «L» и «R» в тексте методики нет — везде только K1, K2, Pr1, Pr2.

Дальше:

- Проводят **не менее двух измерений во времени** (например, в день начала и через ≥15 дней).
- Для первого момента получают **G1**, для второго — **G2** (каждый раз по формуле (1) для своей пары K1, K2, Pr1, Pr2).
- Строят **калибровочную прямую G(D)**:
  - по оси **y** — G (отн. ед.),
  - по оси **x** — давность D (месяцы);
  - прямая имеет **вершину в точке G = 2 при D = 0** и наклон **(G2−G1)/Δt** (Δt — интервал между измерениями в месяцах).
- На этой прямой находят точку с ординатой **G2** и считывают по оси **x** **давность D** (в месяцах).

Из уравнения прямой (вершина в G=2 при D=0) получается:

**D = (G2 − 2) / (G2 − G1)**  
(при Δt = 1 месяц как единице).

То есть **финальную давность можно рассчитать только по K и Pr** (и по двум моментам времени для построения калибровочной прямой), без использования «отдельных» кривых красная/синяя/зелёная — те нужны лишь как сырой сигнал, из которого прибор считает K и Pr.

---

## 6. Выводы для наших графиков

1. **Красная линия** — результирующая амплитуда ССИ; **синяя и зелёная** — реальная и мнимая части (квадратура). Максимумы по красной/синей/зелёной не являются отдельными «метриками» методики; они не подставляются в формулы вместо K или Pr.
2. **K и Pr** в заголовке графика — это и есть те самые параметры методики (из разложения ССИ на короткую и длинную компоненты). Их достаточно для расчёта G и, при наличии двух измерений во времени, для определения D по калибровочной прямой.
3. Для полного расчёта по методике нужны:
   - данные **без покрытия** (K1, Pr1) и **с покрытием** (K2, Pr2);
   - два момента времени (G1 и G2) для построения калибровочной прямой и определения D.
4. В **calculations.pdf** обозначения P1, P2, R1, R2, L1, L2 могут соответствовать тем же K и Pr, но в другой нотации или для двух моментов времени (первое/второе исследование). Формула G = K2/K1 + Pr2/Pr1 в методике одна; расчёт D идёт через калибровочную прямую с вершиной G=2 при D=0.
